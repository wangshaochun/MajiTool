# MajiTool

## 概要

MajiToolは、日常生活や仕事で役立つ便利なツールを集めたWebアプリケーションです。

## 主な機能

- パーセント計算機
- ランダムパスワード生成器

他にも今後さまざまなツールを追加予定です。
